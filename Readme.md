# Live Market Breadth Dashboard 📈

A real-time market breadth tracker for US and Indian Equity markets, built with Python and Streamlit. 

## Features
* Tracks Nifty 500 and S&P 500 / Nasdaq 100.
* Live calculation of stocks above 50, 150, and 200-day SMAs.
* Stage 2 vs Stage 4 trend analysis.
* Sector-wise 52-week high tracking.

## Tech Stack
* **Frontend:** Streamlit & Plotly
* **Data:** yfinance
* **Processing:** Pandas & Numpy
